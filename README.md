<div style="text-align: center;"><h3><a href="#" id="Functions" name="Functions" title="MACD Multi TimeFrame Panel Functions">MACD Multi TimeFrame Panel main functions</a></h3></div>
<p>The <strong>MACD (moving average convergence divergence) MTF Panel indicator</strong> purpose is to:<br/>
– <strong>display the MACD for any timeframe on the current chart, in a small panel</strong><br/>
– <strong>eliminate the need to use the default MACD indicator, therefore allowing more space for the price chart</strong></p>
